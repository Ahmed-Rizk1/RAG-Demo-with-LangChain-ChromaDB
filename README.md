# LangChain RAG Simple Example

This project shows a minimal working setup for a Retrieval-Augmented Generation (RAG) system using LangChain, Chroma, and Streamlit.

![screenshot.png]('screenshot.png)

## Features

- Load local text documents
- Split and embed text using `OpenAIEmbeddings`
- Store and retrieve chunks with Chroma vector store
- Ask questions via Streamlit and get answers using context-aware retrieval

---

## Tech Stack

- LangChain v0.2+
- ChromaDB
- Streamlit
- OpenAI API
- Python 3.11 (not 3.12)
